Hamcrest Matchers: Software Testing Cheat Sheet

Published on: August 30, 2024

Hamcrest Matchers is a Java library that provides a flexible and readable way to write assertions, making it easier to match arrays, collections, numbers, strings, and other objects in tests. It is commonly used for unit testing and works well with JUnit, TestNG, and other testing frameworks. Hamcrest is not bundled by default with JUnit 5 but is included in JUnit 4. In Maven projects, if using JUnit 4, Hamcrest is already available. However, for JUnit 5 or standalone usage, you must explicitly add the Hamcrest dependency in your pom.xml.

Below are a few commonly used Hamcrest assertions:

Matcher Description Example
assertThat(value, equalTo(expected)) Checks if the actual value is equal to the expected value. assertThat(5, equalTo(5));
assertThat(value, not(expected)) Checks if the actual value is not equal to the expected value. assertThat(5, not(4));
assertThat(value, is(expected)) A readable alias for equalTo(). assertThat("hello", is("hello"));
assertThat(value, containsString(substring)) Checks if a string contains a specific substring. assertThat("hello world", containsString("world"));
assertThat(value, startsWith(prefix)) Checks if a string starts with a given prefix. assertThat("Hamcrest", startsWith("Ham"));
assertThat(value, endsWith(suffix)) Checks if a string ends with a given suffix. assertThat("Hamcrest", endsWith("crest"));
assertThat(value, greaterThan(expected)) Checks if the value is greater than the expected value. assertThat(10, greaterThan(5));
assertThat(value, lessThan(expected)) Checks if the value is less than the expected value. assertThat(5, lessThan(10));
assertThat(value, anyOf(matcher1, matcher2)) Passes if at least one matcher condition is met. assertThat(7, anyOf(equalTo(7), greaterThan(10)));
assertThat(value, allOf(matcher1, matcher2)) Passes if all matcher conditions are met. assertThat(7, allOf(greaterThan(5), lessThan(10)));
assertThat(collection, hasSize(size)) Checks if a collection has the expected number of elements. assertThat(Arrays.asList(1,2,3), hasSize(3));
assertThat(collection, hasItem(item)) Checks if a collection contains a specific item. assertThat(Arrays.asList("apple", "banana"), hasItem("banana"));
assertThat(map, hasKey(key)) Checks if a map contains a specific key. assertThat(myMap, hasKey("username"));
assertThat(map, hasValue(value)) Checks if a map contains a specific value. assertThat(myMap, hasValue("admin"));
assertThat(object, hasProperty("property", value)) Checks if an object has a specific property with the expected value. assertThat(user, hasProperty("name", equalTo("Alice"));
assertThat(list, contains(item1, item2)) Checks if a list contains the specified items in the exact order. assertThat(Arrays.asList(1,2,3), contains(1,2,3));
assertThat(list, containsInAnyOrder(item1, item2)) Checks if a list contains the specified items in any order. assertThat(Arrays.asList(3,2,1), containsInAnyOrder(1,2,3));
assertThat(value, instanceOf(Class.class)) Checks if an object is an instance of a specific class. assertThat("hello", instanceOf(String.class));
